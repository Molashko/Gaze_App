# Gaze App

Прототип трекинга взгляда на экране с веб‑камеры. Реализован как GUI‑приложение на Python и использует MediaPipe Face Mesh/Iris.

## Возможности
- Калибровка по сетке 3×3 (9 точек)
- Отслеживание координат взгляда в пикселях экрана
- Полноэкранная визуализация точки взгляда
- Установка зависимостей из GUI

## Требования
- Python 3.11
- Веб‑камера

## Установка
```
python -m pip install -r requirements.txt
```

## Запуск
```
python main.py
```

или

```
python gaze_app.py
```

## Использование
1. Нажмите **«Install Dependencies»** (если не делали вручную).
2. Нажмите **«Calibrate»** и следите за точками калибровки.
3. После завершения калибровки трекинг продолжается автоматически.
4. Для сброса используйте кнопку **«Reset Calibration»**.

## Примечания
- Точность зависит от освещения и стабильности головы.
- При масштабировании Windows (DPI) используются реальные пиксели экрана.
