# Gaze App

Простой прототип трекинга взгляда на экране с веб‑камеры. Реализован как GUI‑приложение на Python и использует библиотеку EyeGestures.

## Возможности
- Калибровка по сетке 5×5 (25 точек)
- Отслеживание координат взгляда в пикселях экрана
- Полноэкранная визуализация точки взгляда
- Установка зависимостей из GUI

## Требования
- Python 3.11
- Веб‑камера

## Установка
```
python -m pip install -r requirements.txt
```

## Запуск
```
python gaze_app.py
```

## Использование
1. Нажмите **«Установить зависимости»** (если не делали вручную).
2. Нажмите **«Калибровка EyeGestures»** и следите за калибровочными точками.
3. После завершения калибровки трекинг продолжается автоматически.
4. Для сброса используйте кнопку **«Сбросить калибровку»**.

## Примечания
- Точность зависит от освещения и стабильности головы.
- При масштабировании Windows (DPI) используются реальные пиксели экрана.
